# 📝 מערכת רישום דיגיטלית - בית ספר טכנולוגי לקיה
### Ali Dallasheh Smart Learning Systems 2026

מערכת רישום חכמה, נגישה ומקצועית שפותחה עבור **המרכז הטכנולוגי לקיה**, המאפשרת להורים לרשום את ילדיהם למגמות הלימוד (חשמל, אוטוטרוניקה ו-CNC) בצורה דיגיטלית מלאה.

---

## 🚀 תכונות המערכת
*   **ממשק דו-לשוני:** תמיכה מלאה בעברית ובערבית (اللقية).
*   **תהליך שלב-אחרי-שלב:** טופס ידידותי המחולק לשלבים (פרטי הורה, פרטי תלמיד, בחירת מסלול).
*   **הטמעת מסמכים:** אפשרות לצילום והעלאת ספח תעודת זהות ישירות מהנייד.
*   **יצירת PDF:** בסיום הרישום, המערכת מפיקה טופס סיכום מעוצב להדפסה או לשמירה כקובץ.
*   **עצמאות טכנולוגית:** כל המשאבים (כולל הלוגו והעיצוב) מוטמעים בתוך קובץ HTML יחיד, מה שמאפשר עבודה ללא תלות בשרתים חיצוניים.

---

## 🛠 טכנולוגיות
*   **HTML5 & CSS3:** עיצוב מודרני מותאם לניידים (Responsive Design).
*   **JavaScript (Vanilla):** ניהול שלבי הטופס, ולידציה של תעודות זהות ועיבוד תמונות בפורמט Base64.
*   **GitHub Pages:** פלטפורמת האירוח להפצת הטופס להורים.

---

## 📋 איך להשתמש?
1.  פתחו את הקישור של האתר.
2.  מלאו את פרטי ההורה והתלמיד (יש להזין 9 ספרות בתעודת הזהות).
3.  בחרו את השכבה והמגמה הרצויה.
4.  צלמו את ספח תעודת הזהות שלכם.
5.  לחצו על "סיים והצג טופס מלא" ולאחר מכן על "הדפס / שמור כ-PDF".
